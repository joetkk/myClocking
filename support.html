myClocking

Public B2B SaaS for any company/organization that needs time & attendance tracking.

Request Access / Create Organization

Any company can request an organization account (not limited to a single company).

Email: support@bca.com.my

Please include:

  • Company/Organization name
  • Contact person name
  • Phone number
  • Email for the admin account
  • Estimated number of employees
  • Country/City

We will reply with organization setup details and admin login instructions.

Support

If you need help troubleshooting:

Email: support@bca.com.my

Please include:

  • Device model (e.g. iPhone 13 / iPad Air)
  • iOS/iPadOS version
  • App version
  • Description of the issue + steps to reproduce
  • Screenshots (if possible)

Testing steps (recommended):

  1. Login as Staff (either demo company).
  2. Go to Check-In / Clock-In.
  3. Allow Location and Camera when prompted.
  4. Submit a check-in and view it in Records/History.
  5. Logout, then login as Tenant/Admin.
  6. Open Records/History (Admin can view all staff records).
